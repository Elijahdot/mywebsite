VionNetwork Web Sitesi Kurulum Talimatları (V2 - Node.js Backend)
=================================================================

Bu proje artık "Dinamik" bir yapıdadır (Node.js + MongoDB + Shopier).
Eski "Sürükle-Bırak" yöntemiyle çalışmaz. GitHub entegrasyonu şarttır.

1. GITHUB'A YÜKLEME
-------------------
1. Bilgisayarınızdaki değişiklikleri kaydedin.
2. Terminali açın ve şu komutları sırasıyla yazın:
   git add .
   git commit -m "Backend guncellemesi"
   git push

2. NETLIFY AYARLARI (ÇOK ÖNEMLİ)
--------------------------------
Siteyi GitHub'a yüklediğinizde Netlify değişikliği görecek ama ÇALIŞMAYACAKTIR.
Çünkü şifrelerinizi girmeniz lazım.

1. Netlify paneline gidin.
2. Sitenizi seçin -> "Site configuration" -> "Environment variables".
3. "Add a variable" butonuna basıp teker teker şunları ekleyin:

   ANAHTAR (Key)          DEĞER (Value)
   ----------------       --------------------------------------------------
   MONGODB_URI            (MongoDB Atlas bağlantı linkiniz)
   JWT_SECRET             (Kafadan uydurduğunuz zor bir şifre)
   SHOPIER_API_KEY        (Shopier panelinden alacağınız API KEY)
   SHOPIER_API_SECRET     (Shopier panelinden alacağınız API SECRET)

3. SONUÇ
--------
Bu ayarları yaptıktan sonra Netlify'da "Deploys" sekmesine gidip 
"Trigger deploy" diyerek tekrar başlatmanız gerekebilir. 
Her şey tamamlandığında site açılacaktır.
